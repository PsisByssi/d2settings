getting started

Dota 2 settings Tweaker helps you get the most out dota without having to waste time messing around with autoexec files
Valve games can often be customized by using the console and entering commands. 

Dota 2 settings tweaker allows you to

Bind actions to any key/joypad/mouse
Macros including setting camera spot to rune positions, using all of the courier abilites 
Access settings that are not currently avaliable in game such as minimap icon size
Better description of obscure options
Snipe heroes, with hp toggler
Test settings for playing around
Easily configure hidden graphics and performance options
